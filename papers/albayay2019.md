# Methods

## Exclusion criteria:
- RTs > 2.5 SD removed (1.8% of trials)
 
## Model
- Task: Go/No-Go with 5:1 or 3:1 Go/NoGo ratio.
- log(RT) ~ 1 + valence + (1 | participant)
- valence is categorical variable with three values (pleasant, unpleasant, control)

## Results
1 significant contrast (p < 0.001)


# Data
- 31 participants
- Dependent variable is RT (mean ~ 335; std ~ 60)
